## 🚀 Quick Start

```bash
# Terminal 1: Start Backend
cd backend
pip install -r requirements.txt
python app.py

# Terminal 2: Start Frontend
cd frontend
npm install
npm start
```

**That's it!** Open http://localhost:3000 🎉
